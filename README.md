# KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a smart relationship management tool designed to help users
track and nurture their connections. It reminds you when it's time to reach out
to friends, ensuring no relationship is left behind.

**🚀 Live Demo**
[**View Live Project**](https://keenkeeperyourlife.netlify.app/)

# 1. Smart Connection Management

- **Dashboard Overview:** Displays total friends, connections on track, and
  those needing urgent attention through summary cards
- **Dynamic Status Logic:** Automatically categorizes friends as `Overdue`,
  `Almost Due`, or `On-Track` based on the last contact date and communication
  goals.

# 2. Dynamic Friend Details

A two-column layout providing deep insights into each connection:

- **Real-time Stats:** Track days since last contact and upcoming due dates.
- **Quick Check-In:** Instantly log interactions (Call, Text, or Video).
- **Interactive Actions:** Professional buttons for Snoozing, Archiving, or
  Deleting connections.

# 3.Interaction Timeline

A dedicated history page that logs every call, text, or video session.

- Automatically updates when a check-in is performed.
- Filterable history to track specific types of communication.

# 4. Friendship Analytics

A visual **Pie Chart** powered by Recharts on the Stats page, showing the
distribution of your communication methods (Call vs. Text vs. Video).

# 5.Technologies Used

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Icons:** React Icons
- **Charts:** Recharts (for Friendship Analytics)
- **Navigation:** Next.js Link & App Router
- **Data Fetching:** Fetch API (from GitHub JSON source)
- **Notifications:** React Hot Toast / Toastify
